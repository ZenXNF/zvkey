## ⚙️ Cara Kerja
Saat install atau update, ZV-Manager akan mengecek apakah IP VPS
terdaftar di repository ini. Jika tidak terdaftar, instalasi akan
ditolak otomatis.

---

## 📋 Format
NamaVPS Expired IPv4
neva-jkt Lifetime    202.155.14.42
client-a 2026-06-01  103.xxx.xxx.xxx
| Kolom | Keterangan |
|---|---|
| NamaVPS | Nama bebas untuk identifikasi |
| Lifetime | Tidak pernah expired |
| YYYY-MM-DD | Tanggal kadaluarsa + 2 hari masa toleransi |
| IPv4 | IP publik VPS |
